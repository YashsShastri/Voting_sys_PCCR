# Secure Online Voting System

A web-based voting system designed for college elections. It ensures secure voting with authentication, role-based access, and controlled revote functionality.

## Features
- Student and Admin login
- Secure voting (one vote per user)
- Revote request with admin approval
- Election management
- Result monitoring
- Email restriction (@pccoer.in)

## Tech Stack
- PHP
- MySQL
- JavaScript
- HTML/CSS

## Setup
1. Install XAMPP
2. Place project in htdocs
3. Import database in phpMyAdmin
4. Start Apache & MySQL
5. Open localhost/project-folder
